Cours de Web

Chapitre 1 : les bases

Les navigateurs permettent de consulter des sites. HTML et CSS sont la base du web -> développer les sites
Créer en 1991
Différencier web et internet.
HTML (hyper text markup language) HTML c’est le contenu (texte, image, lien, etc)
CSS (Cascading Style Sheets, feuille de style) CSS c’est la forme (gérer l’apparence de la page web)
Le CSS repose sur le html
Le WYSIWYG (What you see is what you get)
Doctype : pas une vraie balise, elle sert à dire que c’est bien une page web html écrite en html5.
Balise html : la balise principale du code, elle englobe tout le contenu de la page. (Attention à fermer à la fin)
Head (paire) : Elle sert à mettre certaines informations comme le titre de la page, les formes d’écritures, les lien css, elles ne sont pas affichées sur la page.
Body : corp principale du programme, tout ce qui est dedans sera affiché sur la page web
Meta charset= « utf8 » = C’est un encodage qui permet de déterminer comment certains caractères spéciaux vont s’afficher (impaire)
balise p : paragraphe (paire)
balise br : saut à la ligne
Balise h1, h2, h3, etc. : balise de titre, le plus le numéro est élevé, plus le titre est petit
Balise em (paire) : elle sert à écrire en italique
Balise strong (paire) : elle sert à écrire en gras (utiliser pour afficher des termes importants)
Balise mark : elle sert à surligner les mots (la couleur peut être changé avec le css)
Balise ul (unordered list)(paire) : liste non ordonnée
Balise li (paire) : sert à mettre des points devant des objets de liste
Balise ol (ordered list) (paire) : Remplacer les balises ul par ol (numérote)
Balise dl, dt, dd (impaire) : dl = délimiter la liste, dt = délimiter un terme, dd = délimiter la définition de ce terme
Lien = Texte sur lequel on peut cliquer pour se rendre sur une autre page
Balise a ; attribut href ; valeur de l’attribut est URL de la page ciblée
Exemple : a href="https://google.com" Google/a Une ancre est un lien permettant de se déplacer plus bas dans le site.
Les figures sont des éléments qu’on utilise pour enrichir notre page.
Balise figure (paire) (on peut mettre une/plusieurs images dedans)
On peut ajouter une légende avec la balise figcaption (paire), elle est dans la balise figure

Chapitre 2 : les images

Faire attention au format des images, toutes les pages web ont des images compressées, les machines font des calculs pour que les images soient moins lourdes et plus faciles à afficher.
.jpg ou .jpeg (poids réduit, 16millions de couleurs différentes)
.png (portable network graphics) (plus puissant et récent des formats, l'image peut être rendu transparente et on n'altère pas la qualité de l'image. 2 versions 8bits (256 couleurs) et 24 bits (16M couleurs), la compression du jpg étant plus puissante sur les photos, donc privilégié les jpg).
.gif (image animé)
Ex: img src="nomImage.jpg alt="Image de panda"/
2 attributs obligatoire : src, et alt (description de l'image) Les moteurs de recherches d'images se réfèrent à l'attribut alt.

La miniature cliquable : C'est le resizeimage.net
a href="image/panda.jpg" img src="migniature/panda.jpg" alt="migniature de panda" title="image panda"/a

Les figures :
Éléments qui viennent compléter le texte pour documenter les infos de la page.
figure /figure dans laquelle on peut mettre une image
figcaption /figcaption balise paire à intérieur de la figure pour les citations
